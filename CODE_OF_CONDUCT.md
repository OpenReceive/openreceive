# Code Of Conduct

OpenReceive contributors are expected to be respectful, direct, and generous
with context.

We do not accept harassment, threats, doxxing, sexualized attention, sustained
disruption, or discriminatory behavior. Maintainers may moderate issues,
pull requests, discussions, and project spaces to keep collaboration safe and
productive.

When disagreement is technical, ground it in code, tests, specs, and user
impact. When disagreement is interpersonal, assume good faith once, then ask a
maintainer for help if the pattern continues.
