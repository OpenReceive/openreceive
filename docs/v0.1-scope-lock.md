# v0.1 Scope Lock

v0.1 proves one receive-payment path without expanding into every ecosystem.

## Included

- Repo foundation, governance files, ADRs, and agent rules.
- JSON schemas for invoice, invoice storage lifecycle, payment event, rate
  quote, and error.
- Test vectors for fiat-to-msats, safe integer limits, idempotency, lifecycle,
  settlement detection, polling, and NWC URI redaction.
- NWC client adapter interface.
- Node receive-checkout adapter around `getAlby/js-sdk`.
- Automatic wallet preflight.
- `make_invoice` and `lookup_invoice` wrappers.
- Static price provider.
- Safe poller and settlement detection.
- Express route adapter.
- Browser QR/copy/open-wallet helpers.
- Express + React Hello Fruit demo.
- Manual live-wallet smoke script.
- Basic CI.

## Excluded

- Every language SDK.
- Deterministic mock wallet service.
- Provider wizard and provider-data packages.
- React default UI package.
- Next.js and static HTML demos.
- Ecommerce plugins.
- Native mobile UI kits.
- Edge/serverless live checkout promises.
- Send-payment features in receive-checkout APIs.

## Parallelism Gate

Parallel coding becomes safe only after repo foundation, ADRs, schemas, vectors,
and the NWC client interface exist. Until then, shared contracts stay with one
lead.
